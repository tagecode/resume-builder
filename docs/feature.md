## 功能清单

基于 Electron + shadcn/ui 等技术栈开发跨平台个人简历制作软件，建议功能清单如下：

---

### 一、简历项目管理

- 新建简历（空白 / 从模板创建）
- 简历列表管理（重命名、复制、删除、搜索、排序）
- 多简历切换（支持同时管理多份不同版本）
- 自动保存 & 手动保存（本地持久化）
- 最近编辑记录 & 草稿恢复

---

### 二、智能内容编辑

- **分区编辑模式**（非整页画布式，保证排版稳定）
  - 个人信息（姓名、职位、照片、联系方式、社交链接等）
  - 个人总结 / 求职意向
  - 工作经历（公司、职位、时间、描述，支持增减条目）
  - 教育背景（学校、专业、学位、时间）
  - 技能特长（技能标签、熟练度或分类）
  - 项目经验（项目名、职责、技术栈、成果描述）
  - 证书与奖项
  - 语言能力
  - 自定义模块（用户可增加文本、列表等自定义区块）
- 富文本编辑（加粗、斜体、列表、链接等轻量格式）
- 内容一键排序（拖拽调整区块顺序）
- 模块显示 / 隐藏开关
- 智能提示与填写引导（示例文案）

---

### 三、模板与样式

- 内置多种专业简历模板（按行业、风格分类）
- 模板实时预览（切换模板时内容自动适配）
- 全局样式调节：
  - 字体组合（标题 / 正文字体）
  - 字号与行高
  - 主题色 / 配色方案
  - 页边距、紧凑度（紧凑 / 标准 / 宽松）
- 模块样式微调（如经历条目样式：时间线、卡片式等）
- 自定义模板保存（将当前样式存为用户模板）

---

### 四、实时预览与页面适配

- 分栏式编辑界面（左侧编辑 / 右侧实时预览）
- 预览模式切换（单页 / 双页滚动预览）
- 页数预览与自动分页提示（避免内容溢出）
- 响应式预览缩放
- 暗色 / 亮色预览模式（模拟打印效果）

---

### 五、导入与导出

- **导出为 PDF**（可调纸张大小 A4 / Letter、质量、边距、嵌入字体）
- 导出为图片（PNG / JPEG）
- 打印（直接调用系统打印对话框）
- 导出纯文本 / Markdown 版本
- 导入已有简历（JSON 配置文件、LinkedIn JSON 等，可选）
- 数据导入导出（备份与迁移，JSON 格式）

---

### 六、本地数据与同步（可选）

- 完全本地存储（首次使用无需注册）
- 数据文件自定义存储路径
- 自动备份与历史版本（回退到之前版本）
- 可选云同步（如集成 GitHub Gist、本地文件夹同步、WebDAV）
- 数据导出为标准化 JSON Resume 格式（便于跨平台复用）

---

### 七、辅助与效率功能

- 拼写检查（依赖系统字典或简单集成）
- 字数统计 / 内容长度建议
- 一键优化布局（自动调整间距避免页面孤行）
- 键盘快捷键（保存 Ctrl+S、导出等）
- 多语言界面（中文 / English）
- 内置使用指南 / 引导式入门

---

### 八、桌面端特色（Electron 能力）

- 系统托盘快捷操作
- 全局快捷键唤起 / 隐藏窗口
- 窗口尺寸记忆与位置记忆
- 多窗口支持（同时编辑多份简历并排查看）
- 原生菜单栏集成（文件、编辑、视图、帮助）
- 自动更新检测与静默升级
- 离线完全可用（无需网络）
- 跨平台体验一致（Windows / macOS / Linux）

---

### 九、分享与发布（可选增值）

- 生成在线简历链接（自托管简单服务或导出 HTML 文件）
- 二维码生成（方便手机扫码查看）
- 与招聘平台格式对接（如导出符合特定平台规范的文档）

---

可根据你的目标用户和使用场景，先聚焦基础核心功能（项目管理、内容编辑、模板切换、PDF 导出、本地存储），再逐步扩展高阶能力。
